#MyPalette란?

 Udemy의 Python and Django Full Stack Web Developer Bootcamp 강의 중 CSS Level Two Spectrum Project로 만든 결과물입니다.
팔레트의 색상을 클릭하면 해당 색상을 검색한 구글페이지로 넘어가서 더 다양한 이미지를 볼 수 있습니다.
